# SERIES REQUIRING SPECIAL HANDLING
# =============================================================================

Doctor Who:
  - Must split classic (1963-1989) from modern (2005+)
  - Classic has multi-part serials (e.g., "Part One", "Part Two")
  - TVDB uses different season numbering than some releases
  - Recommendation: Use TVDB ordering, map carefully

Monogatari Series:
  - Complex airing order (Bakemonogatari -> Nisemonogatari -> etc.)
  - Some use "Monogatari Series Second Season" as umbrella
  - Multiple sub-series (Owarimonogatari, Tsukimonogatari, etc.)
  - Recommendation: Use AniDB absolute ordering

One Piece:
  - MASSIVE series (713+ episodes detected so far)
  - Multiple quality/language variants per episode
  - Season numbering differs between sources (TVDB uses different arcs)
  - Recommendation: Use TVDB season mapping, consolidate variants

Legend of the Galactic Heroes:
  - Absolute episode numbering (e.g., "065", "086", "110")
  - Some releases use season notation, others don't
  - Recommendation: Map to TVDB's season structure

Fist of the North Star:
  - Previously managed by Sonarr (has .Sonarr tag)
  - But in wrong folder structure
  - Recommendation: Let Sonarr rename on import
